Overview

LDMud is a virtual machine, a telnet server and a compiler for the LPC language. It is a direct descendent of the original LPMud gamedriver.

Features

Foremost we strive for LDMud to be a stable and clean gamedriver. But furthermore it has also a lot of additional features compared to the original LPMud gamedriver, these are the major ones:

  • Runtime:
    • Support for 64-bit architectures
    • Unicode support including recognition of extended grapheme clusters
  • Language:
    • Advanced type system with union types and type checks at runtime
    • Compound data type named struct
    • Inline closures
    • foreach() statement
  • Networking:
    • Support for TLS
    • Support for IPv6
    • Forwarding of telnet negotiations to the mudlib
    • Support for the MCCP protocol
  • Extensions:
    • Support for efuns written in Python
    • Support for SQLite, MySQL and PostgreSQL
    • Serialization with XML and JSON
